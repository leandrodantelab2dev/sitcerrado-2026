# 🚀 Do Prompt ao Fiori com SAP Build Code

Hands-on prático para criação de aplicações SAP Fiori utilizando IA com o SAP Build Code + Joule dentro do ecossistema SAP BTP Trial.

---

# 📋 Pré-requisitos

Antes de iniciar:

- Notebook com acesso à internet
- Navegador atualizado
- Conta SAP gratuita
- Vontade de brincar com IA dentro do SAP 😎

---

# 🧭 Objetivo do Hands-on

Durante este hands-on você irá:

✅ Criar sua conta SAP  
✅ Provisionar um ambiente Trial no SAP BTP  
✅ Acessar o SAP Build Code  
✅ Utilizar a Joule para geração assistida de aplicação  
✅ Criar uma aplicação SAP Fiori através de prompts  
✅ Entender o fluxo moderno de desenvolvimento SAP com IA

---

# 🔐 Step 1 — Criar Conta na SAP

Acesse:

👉 https://www.sap.com

Crie sua conta gratuita utilizando seu e-mail.

⚠️ Importante:
- Confirme o e-mail enviado pela SAP
- Utilize um e-mail acessível durante o hands-on

---

# ☁️ Step 2 — Acessar o SAP BTP Trial

Após criar sua conta, acesse:

👉 https://cockpit.hanatrial.ondemand.com/trial/#/home/trial

---

# 🌎 Step 3 — Provisionar Ambiente Trial

Ao acessar o Trial:

1. Clique em **Create Trial Account**
2. Escolha o provider:

```txt
AWS
```

3. Escolha a região:

```txt
US East (VA)
```

ou a opção equivalente disponível em EUA.

⚠️ Isso é importante para compatibilidade e disponibilidade dos serviços do hands-on.

---

# 🛠️ Step 4 — Acessar o SAP Build Code

Dentro do BTP Trial:

1. Vá para:

```txt
Subaccount
```

2. Depois:

```txt
Services > SAP Build Code
```

3. Clique em:

```txt
Go To Application
```

---

# 🤖 Step 5 — Criando Aplicação com Joule

Dentro do SAP Build Code:

1. Crie um novo projeto
2. Escolha aplicação Fiori
3. Utilize a Joule para geração assistida
